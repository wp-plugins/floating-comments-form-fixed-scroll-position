Floating Comments allows your users to post comments through a fancy form that fallows them as they scroll down.

== Description ==

This lightweight plugin creates a form that is placed by default ah the left of the content, however you can change its position to anyplace within the screen. The basic function of it is serve as an alternative form to post comments that very handy compared = to the traditional comment form. This form follows you as you scroll down by the page, you can post a comment being at the bottom of the page as well as when on the top.

== Installation ==

1. Upload FloatingComments1.0 to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently asked questions ==

Can I deactivate the fixed position when the form gets to the top?

No, that is not possible, we'll try to implement that option in future versions.
